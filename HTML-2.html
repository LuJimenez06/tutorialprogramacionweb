Etiquetas

Las etiquetas en HTML son los componentes básicos del lenguaje HTML (HyperText Markup Language) que se utilizan para crear y estructurar el contenido en una página web. Cada etiqueta define un elemento dentro de la página y especifica cómo se debe mostrar o comportar ese contenido en el navegador.
Las etiquetas HTML son fundamentales para dar formato, estructurar y definir la semántica del contenido en una página web. Sin etiquetas, un documento HTML no tendría estructura ni significado claro para el navegador o los motores de búsqueda.
Antes de empezar hay que tener en cuenta algo muy importante, dentro de la etiqueta head se pone información importante para la página, pero que no se muestra dentro de la misma, sin embargo, todo lo que pongamos dentro de la etiqueta body si se va a mostrar en la página, con esto aclarado continuemos.

Etiquetas de encabezados

Las etiquetas de encabezado son las etiquetas h1, h2, h3, h4, h5 y h6, estas se utilizan para definir títulos y subtítulos en la página. Las etiquetas h tienen distintos tamaños, siendo h1 la más grande y h6 el más pequeño. A continuación los ejemplos:


<body>

<h1> Texto escrito con H1 </h1>

<h2> Texto escrito con H2 </h2>

<h3> Texto escrito con H3 </h3>

<h4> Texto escrito con H4 </h4>

<h5> Texto escrito con H5 </h5>

<h6> Texto escrito con H6 </h6>

</body>

Etiqueta de espacios

Etiqueta de formato de texto

Las etiquetas de formato de texto en HTML se utilizan para aplicar estilos específicos a partes del contenido textual. Estas etiquetas permiten destacar, enfatizar, o dar estilo a palabras o frases dentro de un documento HTML. A continuación, te presento las más comunes:

strong
La etiqueta <strong> se usa para indicar que un texto tiene una gran importancia o debe ser enfatizado de manera significativa. Generalmente, el contenido dentro de una etiqueta <strong> se muestra en negrita.

b
La etiqueta <b> se utiliza para destacar texto sin añadir ninguna importancia semántica. Es simplemente un cambio visual para hacer que el texto sea negrita. Estilo por Defecto: El texto dentro de <b> se muestra en negrita.

i
La etiqueta <i> se usa para indicar que un texto debe ser resaltado de manera sutil, como por ejemplo para denotar títulos de obras, palabras extranjeras, o nombres científicos. El contenido dentro de <i> se muestra en cursiva.

em
La etiqueta <em> se usa para enfatizar un texto. La diferencia con <i> es que <em> tiene un significado semántico, indicando que el texto es enfatizado. El contenido dentro de <em> se muestra en cursiva.


<p>Este es un <strong>texto importante</strong>.</p>

<p>Este es un texto en <b>negrita</b>.</p>

<p>El término <i>carpe diem</i> es una frase en latín.</p>

<p>Debes <em>prestar atención</em> a este detalle.</p>

Etiqueta de linea horizontal

La etiqueta <hr> en HTML se utiliza para insertar una línea horizontal en un documento. Esta línea generalmente se usa para separar contenido o secciones dentro de una página web.
La etiqueta <hr> se usa para crear una separación visual entre secciones de contenido. Es comúnmente utilizada para dividir temas, indicar un cambio de contexto, o separar bloques de texto.
El elemento <hr> genera una línea horizontal que se extiende a lo largo de su contenedor, por lo general, a lo largo del ancho de la página o del contenedor donde se encuentra. Es una etiqueta de autocierre, lo que significa que no necesita una etiqueta de cierre. Solo se escribe como <hr>. A continuación un ejemplo:


<body>

<p> Los siguientes son espacios <p>
<hr>
<p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Veniam, ipsum alias? Ut minus ipsa pariatur consequatur labore vel veniam voluptatem. Veniam atque, minus ducimus earum modi optio quia amet. Enim? </p>

</body>

Etiqueta de estructura

Las etiquetas de estructura en HTML son fundamentales para organizar el contenido de una página web de manera semántica y lógica. Estas etiquetas ayudan a definir la estructura del documento, mejorando la accesibilidad y la comprensión general del contenido tanto por parte de los usuarios como de los motores de búsqueda. A continuación, te presento las principales etiquetas de estructura:

<header>
La etiqueta <header> se utiliza para definir un encabezado para un documento o una sección. Este encabezado puede contener logotipos, títulos, menús de navegación, y otros elementos introductorios. Generalmente se coloca al inicio de la página o de una sección.

<nav>
La etiqueta <nav> se utiliza para definir un bloque de navegación. Dentro de esta etiqueta se suelen colocar enlaces a otras partes del sitio web, como un menú de navegación. Puede estar dentro de un <header>, en un <aside>, o en cualquier parte de la página donde se necesite un menú de navegación

<aside>
La etiqueta <aside> se utiliza para contenido que es tangencial o complementario al contenido principal. Suele utilizarse para barras laterales (sidebars), información adicional, o anuncios.Generalmente se coloca en un lateral de la página, aunque también puede aparecer dentro de un artículo o sección.

<section>
La etiqueta <section> se usa para definir secciones o bloques temáticos dentro de un documento. Es útil para agrupar contenido relacionado, como un conjunto de artículos o una parte específica de la página. Puede contener otras etiquetas de estructura como <article>, <aside>, etc.

<article>
La etiqueta <article> se utiliza para contenido independiente y autosuficiente que podría ser distribuido o republicado por separado. Esto incluye artículos, entradas de blog, publicaciones en redes sociales, etc. Se ubica dentro de una sección o como parte principal del contenido.

<footer>
La etiqueta <footer> se usa para definir el pie de página de un documento o de una sección. El contenido dentro de <footer> suele incluir información de contacto, derechos de autor, enlaces a políticas de privacidad, y similares. Generalmente se ubica al final de la página o de una sección.
