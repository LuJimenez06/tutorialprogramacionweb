Formato de texto en CSS

CSS permite añadir unas cuantas opciones extras en la personalización de nuestro texto, como agregar otros tipos de letra y para agregar esos nuevos tipos de texto podemos usar el servicio de Google Fonts, lo podemos poner directamente en el HTML o en un CSS

< link href="https://fonts.googleapis.com/css2?family=Gentium+Book+Plus:ital@1&family=Playfair+Display:wght@500&display=swap" rel="stylesheet" >

@import url('https://fonts.googleapis.com/css2?family=Gentium+Book+Plus:ital@1&family=Playfair+Display:wght@500&display=swap');
