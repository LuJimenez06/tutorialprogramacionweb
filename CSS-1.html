Introducción

CSS permite modificar varios archivos (.html) a la vez, asi no tendrás que hacer las distintas configuraciones y solo se necesitará hacer la conección con CSS.
Existen varias opciones para usar CSS, la primera no es tan recomendable pero es una buena opcioón si solo vas a modificar un solo archivo (.html) y no es tan complejo el codigo y es usando la etiqueta "< Styles >":

"< styles >" ... "< /styles >"

Para poder utilizar CSS en HTML, se necesita crear un archivo (.css), y para enlazarlo a HTML se requiere usar la siguiente sentencia:

"< link rel="stylesheet" href="Styles.css" >"

La sentencia se pone debejo de Title.
