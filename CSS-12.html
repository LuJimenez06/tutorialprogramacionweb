Flexbox en CSS

Flexbox es un modelo de diseño en CSS diseñado para distribuir elementos en una página de manera eficiente. Facilita la alineación, el espacio entre elementos y la distribución de elementos dentro de un contenedor flexible, adaptándose a diferentes tamaños de pantalla y resoluciones.

display: flex
El primer paso para usar Flexbox es aplicar display: flex al elemento padre. Esto convierte el contenedor en un contexto de flexbox, permitiendo que sus elementos hijos se comporten de acuerdo con las propiedades de Flexbox.


.container {
  display: flex;
}



Propiedades del Contenedor Flex (Padre)
Estas propiedades se aplican al contenedor que tiene display: flex y controlan la disposición de los elementos flexibles (hijos) dentro del contenedor.

justify-content
Esta propiedad alinea los elementos hijos a lo largo del eje principal (horizontal por defecto) y controla cómo se distribuye el espacio entre los elementos.
Valores comunes:
  • flex-start: Los elementos se alinean al inicio del contenedor.
  • flex-end: Los elementos se alinean al final del contenedor.
  • center: Los elementos se alinean en el centro del contenedor.
  • space-between: Los elementos se distribuyen con espacio igual entre ellos, sin espacio a los lados.
  • space-around: Los elementos tienen espacio igual entre ellos, con espacio reducido en los extremos.
  • space-evenly: Los elementos tienen espacio igual tanto entre ellos como en los extremos.


.container {
  display: flex;
  justify-content: space-between;
}


align-items
Controla la alineación de los elementos a lo largo del eje cruzado (vertical por defecto). Alinea todos los elementos dentro del contenedor de manera uniforme.
Valores comunes:
  • stretch (valor por defecto): Los elementos se estiran para llenar el contenedor.
  • center: Los elementos se alinean verticalmente en el centro.
  • flex-start: Los elementos se alinean al inicio del eje cruzado.
  • flex-end: Los elementos se alinean al final del eje cruzado.


.container {
  display: flex;
  align-items: center;
}


flex-direction
Define la dirección del eje principal, determinando cómo se distribuyen los elementos hijos.
Valores:
  • row (valor por defecto): Los elementos se alinean en una fila horizontal.
  • row-reverse: Los elementos se alinean en una fila inversa.
  • column: Los elementos se alinean en una columna vertical.
  • column-reverse: Los elementos se alinean en una columna inversa.


.container {
  display: flex;
  flex-direction: row-reverse;
}


flex-wrap
Controla si los elementos deben ajustarse a múltiples líneas si no caben en una sola.
Valores:
  • nowrap (valor por defecto): Todos los elementos permanecen en una sola línea.
  • wrap: Los elementos se ajustan en varias líneas.
  • wrap-reverse: Los elementos se ajustan en varias líneas en orden inverso.


.container {
  display: flex;
  flex-wrap: wrap;
}


flex-flow
Es una propiedad abreviada que combina flex-direction y flex-wrap.


.container {
  display: flex;
  flex-flow: column wrap;
}


align-content
Controla el espacio entre las líneas del contenedor flex cuando hay varias líneas de elementos. Solo afecta a los elementos si flex-wrap está habilitado.
Valores:
  • center: Las líneas se agrupan en el centro del contenedor.
  • flex-start: Las líneas se alinean al inicio del contenedor.
  • flex-end: Las líneas se alinean al final del contenedor.


.container {
  display: flex;
  align-content: center;
}



Propiedades de los Elementos Flex (Hijos)
Estas propiedades se aplican a los elementos dentro del contenedor flex y controlan cómo se comporta cada elemento de manera individual.

align-self
Permite alinear individualmente un elemento a lo largo del eje cruzado, sobrescribiendo la propiedad align-items del contenedor.
Valores comunes:
  • auto (valor por defecto): Usa el valor de align-items del contenedor.
  • center: Alinea el elemento en el centro del eje cruzado.
  • flex-start: Alinea el elemento al inicio del eje cruzado.
  • flex-end: Alinea el elemento al final del eje cruzado.
  • stretch: El elemento se estira para llenar el espacio disponible.


.item {
  align-self: center;
}


flex-grow
Define cómo debe crecer un elemento en relación con los demás elementos flexibles. Un valor de 1 significa que el elemento puede crecer si es necesario, mientras que 0 (por defecto) indica que el elemento no crecerá.


.item {
  flex-grow: 1;
}


flex-shrink
Especifica cuánto debe encogerse un elemento en relación con los demás elementos cuando el espacio en el contenedor es limitado. Un valor de 1 permite que el elemento se encoja, mientras que 0 evita que lo haga.


.item {
  flex-shrink: 1;
}


flex-basis
Especifica el tamaño inicial de un elemento antes de que se aplique flex-grow o flex-shrink. Puede ser un valor fijo (px, %) o auto.


.item {
  flex-basis: 200px;
}


flex
Es una propiedad abreviada que combina flex-grow, flex-shrink y flex-basis. Se suele utilizar para definir el comportamiento flexible de los elementos de forma más sencilla.


.item {
  flex: 1 1 100px;
}


order
Define el orden de los elementos dentro del contenedor. El valor por defecto es 0. Un valor mayor moverá el elemento hacia adelante, mientras que un valor negativo lo moverá hacia atrás.


.item {
  order: -1;
}
