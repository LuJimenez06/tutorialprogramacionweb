Animation en CSS

Las animaciones en CSS permiten cambiar de forma dinámica las propiedades de los elementos a lo largo del tiempo. Estas pueden ser usadas para crear transiciones suaves, interactivas y atractivas en las páginas web.

animation
La propiedad animation es un atajo que combina varias propiedades de animación, como animation-name, animation-duration, animation-timing-function, y más. Definirlas correctamente permite personalizar cómo los elementos cambian su apariencia o comportamiento a lo largo del tiempo.


.element {
  margin: 30px;
  width: 120px;
  height: 120px;
  background-color: red;
  border-radius: 50%;
  animation: bounce 2s ease-in-out infinite;
}

Este ejemplo aplica una animación llamada bounce, que tiene una duración de 2 segundos, una curva de tiempo ease-in-out, y se repite indefinidamente.

@keyframes
La regla @keyframes define el conjunto de estilos en diferentes puntos de la animación. Usando porcentajes o palabras clave (from, to), puedes definir el estado inicial, final y cualquier otro punto intermedio.


@keyframes bounce {
  0% { transform: translateY(0); }
  50% { transform: translateY(-100px); }
  100% { transform: translateY(0); }
}




animation-name
Define el nombre de la animación que deseas aplicar al elemento. El nombre debe coincidir con el de los @keyframes.


.element {
  animation-name: bounce;
}


animation-duration
Especifica la duración de la animación. Se puede definir en segundos (s) o milisegundos (ms).


.element {
  animation-duration: 2s;
}


animation-delay
Define el tiempo de espera antes de que la animación comience. Puede ser positivo o negativo (un valor negativo adelanta la animación).


.element {
  animation-delay: 1s;
}


animation-iteration-count
Especifica cuántas veces se repite la animación. Puede ser un número (ejemplo: 3) o infinite para que se repita indefinidamente.


.element {
  animation-iteration-count: infinite;
}


animation-direction
Define la dirección de la animación. Los valores comunes incluyen:
  • normal: La animación se ejecuta en la dirección definida en los keyframes (por defecto).
  • reverse: La animación se ejecuta en sentido inverso.
  • alternate: La animación se alterna entre el sentido normal y el inverso en cada ciclo.
  • alternate-reverse: Alterna la dirección, comenzando por el inverso.


.element {
  animation-direction: alternate;
}


animation-timing-function
Controla la velocidad de la animación durante su duración. Los valores comunes incluyen:
  • ease (valor por defecto)
  • ease-in
  • ease-out
  • ease-in-out
  • linear
  • cubic-bezier(n, n, n, n): Permite crear una curva de velocidad personalizada.


.element {
  animation-timing-function: ease-in-out;
}


animation-fill-mode
Define cómo se aplican los estilos de la animación antes de que comience o después de que termine. Los valores son:
  • none: No aplica ningún estilo extra (por defecto).
  • forwards: Mantiene los estilos finales de la animación.
  • backwards: Aplica los estilos del primer frame durante el tiempo de delay.
  • both: Aplica los estilos tanto al principio como al final de la animación.


.element {
  animation-fill-mode: forwards;
}
