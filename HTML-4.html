Listas en HTML

Las listas en HTML son una manera eficiente de organizar y presentar elementos relacionados de manera estructurada. HTML proporciona tres tipos principales de listas: listas ordenadas, listas no ordenadas, y listas de definición. Cada una tiene un propósito específico y se representa mediante diferentes etiquetas. A continuación, te explico cada una de ellas:


Listas no ordenadas
Las listas no ordenadas se utilizan para presentar elementos sin un orden específico. Cada elemento de la lista está precedido por un marcador o viñeta (bullet). <ul> (unordered list) se usa para definir una lista no ordenada.
Cada elemento dentro de la lista se define con la etiqueta <il> (list item).
El atributo de las listas no ordenadas es type que puede ser (circle), (square), (disc) o (non)


<ul>
  <li> Elemento 1 </li>
  <li> Elemento 2 </li>
  <li> Elemento 3 </li>
</ul>


<ul type="circle">
  <li> Circulo </li>
</ul>

<ul type="square">
  <li> Cuadrado </li>
</ul>

<ul type="disc">
  <li> Disco </li>
</ul>

<ul type="non">
  <li> Sin signo </li>
</ul>

Listas ordenadas
Las listas ordenadas se utilizan cuando los elementos tienen un orden específico. Cada elemento está numerado automáticamente por el navegador. <ol> (ordered list) se usa para definir una lista ordenada.
El atributo de las listas ordenadas es type que puede ser (A), (square), (disc) o (non) Cada elemento dentro de la lista también se define con la etiqueta <il> (list item).

Listas anidadas
Puedes anidar listas dentro de otras listas para crear subcategorías o jerarquías de elementos.


<ul>
  <li> Elemento 1 </li>
  <li> Elemento 2 </li>
  <li> Elemento 3 </li>
    <ul type="circle">
      <li> Circulo </li>
      <li> Circulo </li>
      <li> Circulo </li>
    </ul>
  <li> Elemento 4 </li>
</ul>

Listas de definición
Las listas de definición se utilizan para definir términos y sus descripciones asociadas, como en un diccionario o un glosario.

Etiqueta dl
<dl> (definition list) se usa para definir una lista de definición.

Etiqueta dt
Cada término se define con la etiqueta <dt> (definition term) se usa para definir el termino.

Etiqueta dd
La descripción del término se define con la etiqueta <dd> (definition description) se usa para definir la descripción.


<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado para estructurar páginas web.</dd>
  <dt>CSS</dt>
  <dd>Lenguaje de estilo para diseñar páginas web.</dd>
  <dt>JavaScript</dt>
  <dd>Lenguaje de programación para interactividad en la web.</dd>
</dd>
