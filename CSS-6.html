Estilos de enlace en CSS

Los enlaces son una parte importante de HTML pero su estilo de enlace es muy simple y algo aburrido, pero afortunadamente CSS nos permite darle nuevos estilos, para hacer más dinamicos los enlaces.
Hago una explicación del codigo a traves de notas
Para hacer enlaces más bonitos usaremos un recurso llamado Font Awesome que nos dan simbolos como texto

@import url('https://fonts.googleapis.com/css2?family=Signika:wght@300..700&display=swap');
