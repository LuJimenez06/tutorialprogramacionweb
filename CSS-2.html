Colores

CSS permite añadir colores en texto y en fondos, a diferencia de HTML simple CSS te permite poner muchos más colores





Existen varias opciones para usar colores en CSS:

Poner color

Colores para todo:

Por nombre
Al igual que HTML, CSS permite agregar algunos colores a traves del nombre, pero la lista de CSS es mucho más larga que la de HTML, teniendo más variedad de colores, pero si requieres tonalidades especificas es más recomendable usar las otras opciones, aqui un ejemplo



Por codigo Hexadecimal
Aqui usaremos el codigo Hexadecimal, el codigo Hexadecimal se compone por simbolos que se componen por (1,2,3,4,5,6,7,8,9,A,B,C,D,E,F) de ahí el nombre hexadecimal, porque son 16 caracteres, aqui un ejemplo

Por codigo RGB
El RGB es por Red, Green y Blue, ya que para hacer estos colores se usa un rango de 0 al 255

Por codigo HSL
HSL proviene de "Hue, Saturation and Lightness", que en español sera Matiz, Saturacion y Luminosidad, el matiz maneja un valor de 0 a 360, y la saturacion y la luminosidad de 0 a 100, la saturación marca que tan intenso es el color o que tan gris, mientras que la luminosidad marca que tan claro u oscuro es el colorm aqui unos ejemplos

Colores para fondos:

Por codigo RGBa
El RGBa le añade un canal alfa al RGB, el canal alfa maneja la transparencia del color, esto es muy util cuando se tiene una imagen de fonto y no quieres que el color tape la imagen, el valor del canal alfa va de 0 a 1, 0 es completamente transparente y 1 es el color normal

Por codigo HSLa
El HSLa le añade un canal alfa al HSL, el canal alfa maneja la transparencia del color, el valor del canal alfa va de 0 a 1, 0 es completamente transparente y 1 es el color normal
