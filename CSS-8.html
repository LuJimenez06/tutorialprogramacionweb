Imagenes en CSS

En CSS se puede posicionar las cosas de una forma más dinamica y divertida en comparación de HTML que es muy simple, entre las formas encontramos:

background
La propiedad background es un atajo que permite definir varios aspectos de fondo en una sola línea, como el color, la imagen, la posición, la repetición y el tamaño.

background: #fff url('imagen.jpg') no-repeat center center / cover;

  • #fff: Color de fondo.
  • url('imagen.jpg'): Ruta de la imagen de fondo.
  • no-repeat: La imagen no se repite.
  • center center: La imagen está centrada.
  • cover: La imagen se escala para cubrir todo el área de fondo.

background-color
Esta propiedad define el color de fondo de un elemento.

background-color: #ffcc00; /* Un color amarillo */


background-image
Define una imagen de fondo para un elemento. Usa la función url() para especificar la ubicación de la imagen.

background-image: url('imagen.jpg');


background-repeat
Controla si la imagen de fondo se repite y en qué dirección.
  • no-repeat: No repite la imagen.
  • repeat-x: Repite la imagen solo horizontalmente.
  • repeat-y: Repite la imagen solo verticalmente.
  • repeat: Repite la imagen tanto horizontal como verticalmente.

background-repeat: no-repeat;


background-position
Especifica la posición de la imagen de fondo dentro del elemento. Puedes usar valores como left, right, center, top, bottom, o también valores específicos en píxeles o porcentajes.

background-position: center center;


background-size
Define el tamaño de la imagen de fondo. Los valores comunes son:
  • cover: La imagen cubrirá todo el fondo del elemento, manteniendo su proporción.
  • contain: La imagen se escalará para caber completamente dentro del elemento.
  • También puedes usar valores específicos como 100px 50px.

background-size: cover;


linear-gradient
Un degradado lineal que puedes usar en lugar de una imagen. Puedes definir colores y direcciones.

background: linear-gradient(to right, red, yellow);

  • to right: El degradado va de izquierda a derecha, si no se pone será de arriba hacia abajo.
  • red, yellow: Los colores del degradado.

box-sizing
Define cómo se calculan las dimensiones de un elemento (ancho y alto). Los valores comunes son:
  • content-box: El tamaño incluye solo el contenido, sin el padding o border (valor por defecto).
  • border-box: El tamaño incluye el contenido, padding, y border. Esto es útil para evitar que el padding y el borde agranden el tamaño total del elemento.

box-sizing: border-box;


border-radius
Controla el redondeo de las esquinas de un elemento. Puedes aplicar un valor para todas las esquinas o definir valores individuales para cada esquina (en sentido horario: superior izquierda, superior derecha, inferior derecha, inferior izquierda).

border-radius: 10px; /* Redondea todas las esquinas */

O para esquinas específicas:

border-radius: 10px 0 10px 0; /* Redondea solo las esquinas superiores */
