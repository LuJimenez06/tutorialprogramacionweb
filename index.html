Es un nuevo proyecto para enseñarte cómo hacer una página web, utilizando sus diferentes recursos que ofrece. Estamos creando esta página para ayudar a más personas a entender y comprender cómo crear páginas web, con tutoriales de HTML y CSS.



HTML
Sus siglas significan Lenguaje de marcado de hipertexto, es así el lenguaje estándar más usado para crear y diseñar páginas web. Su mayor característica es estar compuesto por etiquetas que se utilizan para mostrar el contenido de una página web. Con estos tutoriales podrás aprender a crear diferentes elementos como títulos, párrafos, imágenes, enlaces, colocar colores aplicables a tu propio diseño.
HTML al ser tan básico y sin tantas opciones de personalización por lo que nos permite utilizar y conbinarlo con CSS y JavaScript




CSS
Las hojas de estilo en cascada o también conocido como CSS es el lenguaje de estilos que se emplea para describir la presentación de archivos HTML, pero también se usa en XML, SVG, MathML y XHTML.
CSS describe como se carga los elementos en la pantalla.
CSS nos permite crear paginas web más atractivas y llamativas, esté es un "complemento" para HTML.
