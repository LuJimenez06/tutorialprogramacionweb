Grid Layout en CSS

CSS Grid Layout es un sistema de diseño que facilita la creación de diseños complejos y responsivos, utilizando una estructura bidimensional de filas y columnas. Grid permite controlar el tamaño, posición y distribución de los elementos de forma flexible y precisa.

display: grid
Para crear un contenedor con Grid, debes aplicar display: grid al elemento que actuará como el contenedor padre. Esto permite definir la estructura de filas y columnas en las cuales se organizarán los elementos.


.container {
  display: grid;
}



Propiedades del Contenedor Grid (Padre)
Estas propiedades se aplican al contenedor que tiene display: grid y controlan la disposición de las filas, columnas y áreas en las que se organizarán los elementos hijos.

grid-template-columns
Define el número de columnas y sus tamaños dentro del contenedor Grid.
Valores comunes:
  • auto: Las columnas tendrán el tamaño necesario para acomodar el contenido.
  • px: Tamaños fijos en píxeles para cada columna.
  • fr: Una unidad flexible que distribuye el espacio sobrante entre las columnas.
  • repeat(n, tamaño): Repite la definición de columnas n veces.
  • repeat(auto-fill, tamaño): Crea tantas columnas como sea posible, ajustándose al tamaño del contenedor.
  • repeat(auto-fill, minmax(min, max)): Crea columnas que se ajustan automáticamente dentro de un rango mínimo y máximo.


.container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
}


grid-template-rows
Define el número de filas y sus tamaños dentro del contenedor Grid. Funciona de manera similar a grid-template-columns.


.container {
  grid-template-rows: repeat(3, 200px);
}


column-gap
Controla el espacio entre las columnas.


.container {
  column-gap: 20px;
}


row-gap
Controla el espacio entre las filas.


.container {
  row-gap: 20px;
}


gap
Es una propiedad abreviada que combina row-gap y column-gap. Controla el espacio entre filas y columnas de manera simultánea.


.container {
  gap: 20px;
}


align-items
Controla la alineación vertical de los elementos dentro del contenedor Grid.
Valores comunes:
  • center: Alinea los elementos en el centro vertical del contenedor.
  • start: Alinea los elementos en la parte superior del contenedor.
  • end: Alinea los elementos en la parte inferior del contenedor.


.container {
  align-items: center;
}


justify-items
Controla la alineación horizontal de los elementos dentro del contenedor Grid.
Valores comunes:
  • center: Alinea los elementos en el centro horizontal del contenedor.
  • start: Alinea los elementos al principio del contenedor (izquierda).
  • end: Alinea los elementos al final del contenedor (derecha).


.container {
  justify-items: start;
}


grid-template-areas
Define áreas nombradas dentro del Grid, lo que facilita organizar elementos en lugares específicos dentro de la cuadrícula.


.container {
  grid-template-areas:
    'header header header article'
    'sidemenu main main article'
    'sidemenu main main article'
    'footer footer footer footer';
}

Los elementos hijos se asignan a estas áreas con grid-area.


Propiedades de los Elementos Grid (Hijos)
Estas propiedades controlan el posicionamiento y tamaño de los elementos hijos dentro del contenedor Grid.
grid-column-start y grid-column-end
Controlan el inicio y el final de un elemento en el eje de las columnas. Especifican desde qué columna comienza y hasta dónde se extiende.


.item {
  grid-column-start: 1;
  grid-column-end: 3;
}


grid-column
Es una propiedad abreviada que combina grid-column-start y grid-column-end.


.item {
  grid-column: 1 / 3;
}

También puedes usar span para que el elemento abarque varias columnas:


.item {
  grid-column-start: 1;
  grid-column: span 2;
}


grid-row-start y grid-row-end
Controlan el inicio y el final de un elemento en el eje de las filas. Funcionan de manera similar a las propiedades de columna.


.item {
  grid-row-start: 1;
  grid-row-end: 2;
}


grid-row
Es una propiedad abreviada que combina grid-row-start y grid-row-end.


.item {
  grid-row: 1 / 2;
}

También puedes usar span para que el elemento abarque varias columnas:


.item {
  grid-row-start: 1;
  grid-row: span 2;
}


grid-area
Es una propiedad que permite posicionar un elemento en un área específica del Grid definida con grid-template-areas. También puede combinar grid-row y grid-column para especificar el área en términos de líneas de la cuadrícula.


.item {
  grid-area: header;
}

Ejemplo usando líneas de la cuadrícula:


.item {
  grid-area: 1 / 1 / 2 / 3; /* Fila inicio / Columna inicio / Fila fin / Columna fin */
}
