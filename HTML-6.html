Tablas

Las tablas en HTML se utilizan para organizar y presentar datos en un formato tabular, con filas y columnas. Aunque no son tan comúnmente utilizadas para el diseño de páginas web hoy en día, debido a la adopción de CSS para ese propósito, aún siguen siendo muy útiles para mostrar información estructurada, como horarios, listas de precios, y comparaciones de productos.


Estructura Básica de una Tabla en HTML
La estructura de una tabla en HTML se define utilizando varias etiquetas específicas:
<table> (table)
La etiqueta <table> es el contenedor principal para definir una tabla.
<caption>
La etiqueta <caption> define el nombre de la tabla.
<tr> (Table Row)
La etiqueta <tr> define una fila en la tabla.
<th> (Table Header)
La etiqueta <th> define una celda de encabezado en la tabla. Por defecto, el texto en las celdas <th> se muestra en negrita y centrado.
<td> (Table Data)
La etiqueta <th> define una celda de datos en la tabla. Estas celdas contienen el contenido real de la tabla.

Un ejemplo completo de una tabla simple podría verse así:


<table border="1">
  <tr>
    <th> Nombre </th>
    <th> Edad </th>
    <th> Ciudad </th>
  </tr>

  <tr>
    <td> Ana </td>
    <td> 28 </td>
    <td> Madrid </td>
  </tr>

  <tr>
    <td> Carlos </td>
    <td> 35 </td>
    <td> Barcelona </td>
  </tr>
</table>


Atributos de la Tabla
border
Controla el grosor del borde de la tabla. Aunque este atributo se usa menos hoy en día, ya que CSS es la forma recomendada de manejar los estilos, todavía puede ser útil.
cellpadding y cellspacing
cellpadding controla el espacio interno dentro de las celdas, mientras que cellspacing controla el espacio entre las celdas.


<table border="1" cellpadding="10" cellspacing="5">
  <tr>
    <th> Nombre </th>
    <th> Edad </th>
    <th> Ciudad </th>
  </tr>

  <tr>
    <td> Ana </td>
    <td> 28 </td>
    <td> Madrid </td>
  </tr>

  <tr>
    <td> Carlos </td>
    <td> 35 </td>
    <td> Barcelona </td>
  </tr>
</table>


Combinación de Celdas
colspan
Permite que una celda se extienda por varias columnas.
rowspan
Permite que una celda se extienda por varias filas.


<table border="1">
  <tr>
    <th colspan="2"> Encabezado que ocupa dos columnas </th>
  </tr>
  <tr>
    <td> Dato 1 </td>
    <td> Dato 2 </td>
  </tr>
</table>


<table border="1">
  <tr>
    <td rowspan="2"> Celda que ocupa dos filas </td>
    <td> Dato 1 </td>
  </tr>
  <tr>
    <td> Dato 2 </td>
  </tr>
</table>


Subdivisión de la Tabla
<thead>
Define un grupo de filas que representan el encabezado de la tabla.
<tbody>
Agrupa el contenido principal de la tabla, es decir, las filas de datos.
<tfoot>
Define un grupo de filas que representan el pie de la tabla. Suele utilizarse para sumarios o totales.


<table border="1">
  <thead>
    <tr>
      <th> Nombre </th>
      <th> Edad </th>
      <th> Ciudad </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td> Ana </td>
      <td> 28 </td>
      <td> Madrid </td>
    </tr>
    <tr>
      <td> Carlos </td>
      <td> 35 </td>
      <td> Barcelona </td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="3"> Total de registros: 2 </td>
    </tr>
  </tfoot>
</table>
