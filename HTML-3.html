Enlaces de HTML

Los enlaces en HTML son fundamentales para la navegación en la web, ya que permiten conectar diferentes páginas o recursos dentro de un sitio web, así como dirigir a los usuarios a sitios externos. La etiqueta principal utilizada para crear enlaces es <a>, que significa "anchor" (ancla). A continuación, te doy una explicación detallada sobre los enlaces en HTML.
La etiqueta <a> se usa para crear un enlace, que puede apuntar a otra página web, un archivo, una sección específica de la misma página, o incluso iniciar una llamada telefónica o enviar un correo electrónico. Para que el enlace no te envie a ningún lugar se pone el numeral o gato como valor en el atributo href. href="#"


Atributo href y tipos de enlaces

El atributo href (hypertext reference) es el más importante dentro de la etiqueta <a>. Define la URL del destino al que llevará el enlace.

Enlaces a páginas externas
Son las que dirigen a páginas que no pertenecen al mismo dominio. Para que funcione correctamente el enlace se tiene que poner completo, sin omitir el "http://wwww" o el "https://www"

Enlaces a páginas internas
Son las que dirigen a páginas que pertenecen al mismo sitio web. Si quieres hacer un enlace a un archivo que se encuentra en una carpeta se pone asi: href="(nombre de la carpeta)/(nombre del archivo)"

Enlaces a secciones especificas de la misma página
Son las que dirigen a secciones especificas que utiliza un identificador (ID) para saltar a una parte específica de la misma página. Se usa la configuracion <i id="seccion"> sección especifica </i>

Enlaces para enviar correos electrónicos
Son las que dirigen al correo electrónico. Utiliza el esquema mailto: para abrir el cliente de correo electrónico predeterminado del usuario.

Enlaces para llamadas telefónicas
Son las que dirigen a la llamada. Utiliza el esquema tel: para iniciar una llamada en dispositivos móviles.

Atributo target

El atributo target se usa para definir dónde se abrirá el enlace.
  • _self: Abre el enlace en la misma pestaña o ventana. Es el valor predeterminado.
  • _blank: Abre el enlace en una nueva pestaña o ventana.


<a href="https://www.youtube.com/" target="_blank"> Abrir en una nueva pestaña </a>



Abrir en una nueva pestaña



Atributo rel

El atributo rel define la relación entre la página actual y la página a la que apunta el enlace. rel="noopener noreferrer": Se usa junto con target="_blank" para mejorar la seguridad y el rendimiento.
