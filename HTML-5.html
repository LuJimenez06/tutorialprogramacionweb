Imágenes

Las imágenes en HTML son un elemento clave para enriquecer el contenido visual de una página web. Se utilizan para mostrar gráficos, fotos, logotipos, íconos, y más.
La etiqueta <img> en HTML soporta varios formatos de imagen, cada uno con sus propias características y casos de uso. Los formatos más comunes que se pueden utilizar con la etiqueta <img> son: jpg, jpeg, png, gif, webp... pero tambien permite formatosde imagenes vectoriales como svg, y formatos de imagenes especializados como bmp y avif, y los antiguos tiff y tif.
A continuación, te proporciono una explicación completa sobre cómo se manejan las imágenes en HTML.


Etiqueta img
La etiqueta <img> se utiliza para incrustar imágenes en un documento HTML. Es una etiqueta auto-cerrada, lo que significa que no necesita una etiqueta de cierre. La etiqueta img tiene dos atributos base, y estos son:

Atributo src
El atributo src (source) es obligatorio y especifica la URL de la imagen que se va a mostrar. La URL puede ser relativa o absoluta. Una imagen es relativa si la imagen está en el mismo servidor, se puede invocar por medio del nombre del archivo o en caso de que esté en una carpeta por medio de la ruta de la carpeta y el nombre del archivo. En cambio, una imagen es absoluta si la imagen está en otro servidor, se puede invocar por medio del url de la imagen.

Atributo alt
El atributo alt (alternative text) es obligatorio y proporciona una descripción alternativa de la imagen si no se puede cargar. Además, es esencial para la accesibilidad web, ya que los lectores de pantalla utilizan este texto para describir la imagen a usuarios con discapacidades visuales.

Ahora los ejemplos con una imagen relativa y una imagen absoluta:
Imagen relativa:

<img src="ola.jpg" alt="ola">



<img src="https://as2.ftcdn.net/v2/jpg/02/13/61/87/1000_F_213618717_OtjtaJR6zOBtZO1QKUHrcGS1TTXWGxqV.jpg" alt="El mar">




Atributos width y height
Los atributos width (ancho) y height (alto) se utilizan para especificar las dimensiones de la imagen. Los valores se pueden definir en píxeles (px) o en porcentajes (%).

<img src="ola.jpg" alt="ola" width="600px" height="400">




Atributo title
El atributo title se utiliza para proporcionar un texto emergente que aparece cuando un usuario pasa el ratón sobre la imagen. Este fucniona como el titulo de la imagen

<img src="ola.jpg" alt="ola" title:"Ola en el mar">




Imagenes como enlaces
Puedes hacer que una imagen actúe como un enlace utilizando la etiqueta <a>.

<a href=" https://bit.ly/CodingHTML01"><img src="html5.png" alt="HTML5" title:"Logo de HTML5"></a>




Mapas de Imagen (<map> y <area>)
Un mapa de imagen te permite definir áreas específicas dentro de una imagen que son clicables, dirigiendo al usuario a diferentes enlaces.
<map>: Enlaza la imagen con el mapa de imagen definido.
<area>: Define las áreas clicables, con atributos como shape (forma), coords (coordenadas), y href (enlace).


<img src="html5.png" alt="HTML5" usemap="#mapa-ejemplo">
<map name="mapa-ejemplo">
  <area shape="circle" coords="18,18,342,342" href="https://bit.ly/CodingHTML01" alt="Enlace 2">
  <area shape="rect" coords="0,0,360,360" href="https://www.youtube.com/@CodingTube" alt="Enlace 1">
</map>




Atributo loading
El atributo loading controla la carga diferida de imágenes, lo que puede mejorar el rendimiento de la página. <a>.
lazy: Retrasa la carga de la imagen hasta que esté a punto de aparecer en la pantalla.
eager: Carga la imagen de inmediato.
