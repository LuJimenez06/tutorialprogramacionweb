Transform en CSS

Las transformaciones en CSS permiten cambiar la apariencia y posición de un elemento sin alterar su flujo en el documento. Con la propiedad transform, puedes aplicar varias transformaciones visuales, como traslaciones, rotaciones, escalados y sesgados, tanto en 2D como en 3D.

transform
La propiedad transform es la clave para realizar las transformaciones de un elemento. Se utiliza para aplicar una o varias transformaciones como translate, rotate, scale, etc.


.element {
  transform: translateX(100px) rotate(45deg);
}


transform-origin
Define el punto de origen para la transformación de un elemento. Por defecto, este origen está centrado en el elemento (50% 50%), pero puedes cambiarlo.


.element {
  transform-origin: top left;
}

Esto moverá el punto de rotación o escalado al borde superior izquierdo del elemento.

perspective
perspective se utiliza para crear una ilusión de profundidad en transformaciones 3D, definiendo qué tan cerca o lejos está el punto de vista. Se suele aplicar al contenedor que alberga los elementos que serán transformados en 3D.


.container {
  perspective: 1000px;
}


Funciones 2D
Estas funciones permiten aplicar transformaciones en un plano bidimensional.

translate()
Desplaza el elemento en los ejes X e Y. Los valores positivos mueven el elemento hacia la derecha o abajo, y los negativos, hacia la izquierda o arriba.


.element {
  transform: translate(50px, 100px);
}


rotate()
Rota el elemento alrededor de su punto de origen. El valor es el ángulo en grados.


.element {
  transform: rotate(45deg);
}


scale()
Escala el tamaño del elemento. Los valores menores que 1 lo reducen, y mayores que 1 lo aumentan.


.element {
  transform: scale(1.5);
}


skew()
Sesga o inclina el elemento en el eje X o Y.


.element {
  transform: skew(20deg, 10deg);
}


matrix()
Es la función más compleja de transformaciones 2D y permite combinar todas las anteriores en una sola matriz de transformación.


.element {
  transform: matrix(1, 0.5, 0.5, 1, 10, 20);
}



Funciones 3D
Las transformaciones 3D permiten dar profundidad y rotar los elementos en tres dimensiones.

rotateX()
Rota el elemento alrededor del eje X (horizontal).


.element {
  transform: rotateX(45deg);
}


rotateY())
Rota el elemento alrededor del eje Y (vertical).


.element {
  transform: rotateY(45deg);
}


rotateZ()
Rota el elemento alrededor del eje Z (similar a rotate() en 2D).


.element {
  transform: rotateZ(45deg);
}
