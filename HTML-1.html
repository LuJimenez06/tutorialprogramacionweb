Introducción

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar contenido en la web. Con HTML, puedes definir la estructura de una página web utilizando diferentes tipos de elementos como encabezados, párrafos, imágenes, enlaces, y más.


¿Qué son las etiquetas?
Las etiquetas en HTML son los elementos fundamentales que se utilizan para definir y estructurar el contenido en una página web. Cada etiqueta está rodeada por corchetes angulares (< y >), y la mayoría de las etiquetas vienen en pares: una etiqueta de apertura y una etiqueta de cierre. La etiqueta de cierre se diferencia de la de apertura porque incluye una barra inclinada (/) antes del nombre de la etiqueta. Ejemplo: <p> - texto - </p>


Estructura Básica de un Archivo HTML
Un archivo HTML es esencialmente un documento de texto que utiliza etiquetas para definir y organizar el contenido. A continuación, te muestro cómo se estructura un documento HTML básico:

< !DOCTYPE html >
< html lang="es" >
< head>"
  < meta charset="UTF-8" >
  < meta name="viewport" content="width=device-width, initial-scale=1.0" >
  < title > Título de la Página < /title >
< /head >"
< body >"
  < !-- Contenido de la página va aquí -- >
< /body >
< /html >

  • < !DOCTYPE html >: Esta declaración define el tipo de documento y la versión de HTML que estás utilizando.
  • < html >: La etiqueta html envuelve todo el contenido de la página web.
  • < head >: Dentro de head se incluyen metadatos, el título de la página, enlaces a archivos CSS, y scripts.
  • < title >: Define el título que aparecerá en la pestaña del navegador.
  • < body >: Todo lo que quieras mostrar en la página web debe estar dentro de .

Creación y Organización de un Archivo HTML
Para empezar a crear tu propio archivo HTML, sigue estos pasos:
  1. Crea un nuevo archivo: Usa un editor de texto como Notepad++, Visual Studio Code o cualquier otro que prefieras. Guarda el archivo con la extensión .html.
  2. Escribe el código básico: Utiliza la estructura básica que te mostré anteriormente como punto de partida.
  3. Agrega contenido: Empieza a insertar etiquetas HTML para añadir encabezados, párrafos, imágenes, y cualquier otro contenido que desees.
  4. Visualiza en el navegador: Abre el archivo .html en tu navegador para ver cómo se renderiza tu página web.


Comentarios en HTML
Para hacer un comentario en HTML utilizamos la estructura "<!----- (Texto)>", los comentarios sirven para ayudarnos a nosotros como programador o a las personas que leen nuestro código, teniendo una breve descripción. Los comentarios no se ejecutan ni se muestran en la página web.
